# Smart Life Hub (智慧生活控制中心)

這是一個基於 React + Vite 開發的多功能生活入口應用程式 (PWA)，旨在為使用者提供一站式的個人生活管理體驗。所有資料均儲存於瀏覽器本地 (Local Storage)，確保隱私且支援離線使用。

## 🌟 核心特色
- **PWA 支援**：可安裝至手機主畫面，體驗接近原生 App。
- **隱私優先**：不設後端資料庫，資料完全留在使用者的設備中。
- **四種美學主題**：
  - 🎨 **可愛卡通 (Cartoon)**：柔和色彩與圓角設計。
  - 💼 **專業嚴肅 (Serious)**：深色背景與高對比，穩定且清晰。
  - 🌿 **清新森林 (Forest)**：大自然色彩，平靜且舒壓。
  - ◻️ **極簡黑白 (Minimalist)**：純粹的線條與對比。
- **安全性**：支援 4 位數 PIN 碼「程式鎖」，防止他人窺探隱私。
- **資料管理**：支援 JSON 格式的備份匯出與還原功能。

## 📱 16 個子應用詳解

### 1. 英文學習 (English Learning)
- **核心功能**：每日推薦 10 個核心單字。
- **單字難度**：支援「基礎 (1000單)」與「進階 (7000單)」切換。
- **技術細節**：單字源於靜態詞庫，結合本地狀態追蹤每日進度。

### 2. 加油紀錄 (Fuel Tracker)
- **核心功能**：記錄愛車的加油週期、油耗表現、保養紀錄與花銷。
- **特色**：支援多車型記錄 (汽車/機車)，自動計算行駛里程，整合「保養日誌」功能。

### 3. 信用卡費 (Credit Card)
- **核心功能**：統整多張信用卡的額度與各月帳單金額。
- **特色**：視覺化顯示信用額度佔用率，追蹤每月的消費波動。

### 4. 運動紀錄 (My Sport)
- **核心功能**：串接 **Strava API** 獲取最新的跑步、單車運動數據。
- **技術細節**：使用 OAuth 2.0 授權，自動同步最近 30 條活動紀錄。

### 5. YouBike 即時站點
- **核心功能**：全台 YouBike 2.0 站點即時車位查詢。
- **特色**：支援雙北、桃園、台中等 10+ 個城市，顯示可借、可還與電輔車數量。
- **數據來源**：TDX 運輸資料流通服務平臺 (V2 API)。

### 6. 台鐵時刻表 (TRA Schedule)
- **核心功能**：快速查詢台鐵班次時刻。
- **特色**：
  - **智慧過濾**：自動顯示基準時間前 30m 至後 2h 的班次。
  - **即時誤點**：顏色燈號顯示（準點/誤點資訊）。
- **數據來源**：TDX 運輸資料流通服務平臺 (V3 & V2 API)。

### 7. 排班管理 (Shift Management)
- **核心功能**：針對服務業或排班人員設計的行事曆工具。
- **特色**：
  - 支援自訂人員與班別（顏色與名稱）。
  - **快速編輯**：直接點擊人員即可修改資料（含電話）。
  - **一鍵派人**：在今日概括中快速檢視並撥打當班人員電話。

### 8. ToDo 清單 (ToDo List)
- **核心功能**：任務管理與分類清單。
- **特色**：採用優雅的文學風格背景，支援分類顏色標註與完成狀態切換。

### 9. 抽籤轉盤 (Lucky Draw)
- **核心功能**：解決選擇困難症的神器。
- **包含功能**：自訂轉盤、隨機數字產生器、拋硬幣動畫。
- **視覺效果**：支援全屏動畫與互動音效感。

### 10. 個人日記 (Diary)
- **核心功能**：私密的圖文記事本。
- **特色**：支援心情標籤、天氣記錄與各類主題字體切換。

### 11. 體重紀錄 (Weight Tracker)
- **核心功能**：追蹤體重變化，計算 BMI 與理想體重目標。
- **特色**：記錄歷史體重曲線，提供建議的周減重目標。

### 12. 簡單看盤 (Stock Tracker)
- **核心功能**：關注台股與美股三大指數即時走勢。
- **技術細節**：串接 **Alpha Vantage API**，提供自訂自選股清單功能。

### 13. 答案之書 (Answer Book)
- **核心功能**：在迷惘時提供冥想式的隨機指引。
- **特色**：精緻的翻頁動畫，結合內在平靜的導引流程與多層次的答案詮釋。

### 14. 極簡匯率 (Exchange App)
- **核心功能**：即時全球匯率轉換。
- **特色**：支援多國貨幣快速切換，提供極簡、無廣告的數據查詢體驗。

### 15. 停車場在哪 (Parking App)
- **核心功能**：即時查詢全台停車場剩餘車位。
- **數據來源**：TDX 運輸資料流通服務平臺 (V1 API)。

### 16. 電影院 (Movie App)
- **核心功能**：瀏覽全台「現正熱映」與「即將上映」電影。
- **技術細節**：串接 **TMDB API**，提供 4K 高品質海報、評分與劇情摘要。

## 🛠️ 開發架構與技術棧

- **Frontend**: React (Hooks, Context API)
- **Build Tool**: Vite
- **Styling**: Vanilla CSS + Tailwind-like custom logic
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **State Management**: Zustand + Persistence Middleware
- **Routing**: React Router DOM

## 📥 安裝與運行

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建立正式生產版本
npm run build
```
